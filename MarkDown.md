## Списки

Чтобы добавить не нумерованный список необходимо элементы выделить звездочкой с одной стороны:

* Элемент 1
* Элемент 2
* Элемент 3
+ Элемент 4


Чтобы добавить нумерованный список нужно поставить номер с точкой

1. Элемент 1
2. Элемент 2
3. Элемент 3
